EGFブースターとは?

EGFブースターはFGFおよび複数の成長因子を含む線維芽細胞培養液で、 保湿、抗酸化、老化防止、肌再生、しわ改善に効果的です。

コラーゲン、エラスチン、ヒアルロン酸などの成分が細胞生成を促進し、 肌再生と肌バリアを強化し、アンチエイジングをサポートします。

0.25mmのMTS(マイクロニードル)に塗布して施術し、 肌に数百の微細な穴を作り、肌の深い部分まで有効成分を届け、 LDMの超音波導入モードを通じて吸収率を高めます。

ハリ
線維芽細胞の成長を助けるFGFを配合

150,000PPMの線維芽細胞培養液が 肌のエラスチンとコラーゲンの形成を助け、 肌のハリ改善効果を生み出します。

美白 3種の植物幹細胞の抗酸化力 ビタミンCの2倍高い抗酸化効果で、 紫外線から肌を守り、 肌の再生を助けます。

線維芽細胞培養液は脂肪幹細胞培養液とは異なり、

脂肪ではなく皮膚組織から細胞を採取します。

単にコラーゲンを生成するだけの一般的な幹細胞培養液ではなく、 老化したコラーゲン線維組織を分解し、真皮を新しいコラーゲンで満たして、 ハリのある肌を作り上げます。

LDMの薬物吸収モードによる2次施術を通じて

最高のEGFブースター効果を引き出します。

AMTS 数百本のマイクロニードルで 表皮に微細な穴を開け、 線維芽細胞培養液がよく吸収されるよう助けます。

LDM 超音波導入(Sonophoresis)モードを通じて 肌の薬物吸収率を高めます。

( 01 )

1:1肌コンディション＆デザインカウンセリング

( 02 )

韓方医院長2人による
特別な専任ケア

カウンセリングから施術まで、家族や大切な人の肌だと思い、誰よりも丁寧できめ細やかにケアします。 肌の美容に韓方医学の視点を取り入れ、血脈と筋肉への刺激を加えることで、健康と美しさの両方を実現します。

( 03 )

副作用のない健康的な美しさ

( 04 )

適正量の施術
